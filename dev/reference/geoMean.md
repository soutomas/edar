# Geometric mean

Compute geometric mean.

## Usage

``` r
geoMean(x, ...)
```

## Arguments

- x:

  `<num>` A vector of values.

- ...:

  Other augments to pass to
  [`mean()`](https://rdrr.io/r/base/mean.html).

## Value

Geometric mean.

## Examples

``` r
geoMean(rlnorm(10))
#> [1] 0.6431582
```
