# Generate time stamp label

Generate a label with a time stamp indicating the run time.

## Usage

``` r
label_tz(omit = "")
```

## Arguments

- omit:

  `<chr>` Text to omit from the label.

## Value

A label with time stamp.

## Examples

``` r
label_tz()
#> [1] "Run: 2025-12-02 19:06:35 UTC"
```
